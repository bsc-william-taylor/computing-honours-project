Basics

Raster is a JavaScript runtime for GPU centric applications. Through raster you can access the latest OpenGL and OpenCL standards for a feature full playground to develop GPU techniques and or GPU based applications such as games.

Raster has a fully featured standard library with utilities to handle windows, http requests, console, io and much more as well as native bindings to OpenCL 2.0 and OpenGL 3.3 & 4.5

What's more the entire runtime can be embeded into a C++ app allowing you to export your raster app as a standalone executable if you wish.


Raster is currently in development.
